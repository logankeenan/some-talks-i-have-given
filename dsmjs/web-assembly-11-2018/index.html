What is Web Assembly?

WebAssembly (abbreviated Wasm) is a binary instruction format for a stack-based virtual machine.

It's fast, safe, open, and part of the web platform. Currently, version 1.0 has shipped in all 4 major browsers

How can we write/compile Web Assembly?

Web Assembly can be the compiled target for a variety of languages.

Rust

Rust is a systems programming language that runs blazingly fast, prevents segfaults, and guarantees thread safety. Source

Rust is currently the most active community support Web Assembly as a compile target. Github

Getting started with Rust

  1. Install: 
    curl https://sh.rustup.rs -sSf | sh
  2. Add emscripten compile target 
    rustup target add wasm32-unknown-emscripten
  3. Setup Emscripten 
    curl https://s3.amazonaws.com/mozilla-games/emscripten/releases/emsdk-portable.tar.gz | tar -zxv -C ~/ 
    cd ~/emsdk-portable
./emsdk update
./emsdk install sdk-incoming-64bit
./emsdk activate sdk-incoming-64bit
